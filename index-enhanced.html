⚡ Quick Reference

Parametric Rolling: Frequency ratio 1.8-2.2× (wave period 10-20s)

Risk: Exponential amplitude growth | Action: Speed change ±2-3 knots

Synchronous Rolling: Frequency ratio ≈1.0× (wave period 18-22s)

Risk: Gradual amplitude growth | Action: Speed change ±1-2 knots

📱 How to Get Started

1. Configure Your Ship: Click ⚙️ Config → Enter ship parameters or load preset → Save

2. Use Monitor: Click 📊 Parametric or 🌊 Synchronous → Input sea state → Monitor readings

3. Log Observations: Scroll down → Add note → Data saves automatically

4. Export Logs: Bottom of log section → "Export CSV" → Downloads to iPhone

5. Switch Monitors: Use navigation menu at top → No page reload
